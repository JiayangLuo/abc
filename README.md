# 基于多层膜系统的空间微分器模拟程序 (Spatial Differentiator Based on Multilayer Film System)
## 简介
这是一个基于 MATLAB 的多层膜系统的空间微分器模拟程序，用于计算和分析多层膜结构中的光学特性，包括反射率、透射率和电场分布等，通过调节多层膜的结构，入射光经过多层膜结构后，可实现对入射光场的空间微分。
## 系统结构
该程序模拟的多层结构包含：
- 顶层：空气
- 第一层：MoS2 (厚度 0.65nm)
- 第二层：SiO2 (厚度 300nm)
- 底层：硅基底
## 主要功能
1. **反射率计算**
   - 单层界面反射率计算
   - 多层结构反射率计算
   - S偏振和P偏振的分别计算
2. **场分布模拟**
   - 入射场计算
   - 反射场计算
   - 电场分量分析
3. **可视化功能**
   - 反射率随波长和入射角的变化
   - 电场分布图
   - 相位分布图
## 使用方法
1. 确保系统中已安装 MATLAB
2. 将所有代码文件放在同一目录下
3. 运行主程序 `mulOpt.m`
4. 程序将自动计算并显示结果

## 参数设置
```matlab
% 基本参数
lambda = 400:1:800;      % 波长范围：400-800nm
ang0 = 0:0.1:90;         % 入射角度范围：0-90度

% 结构参数
d1 = 0.65;               % MoS2层厚度(nm)
d2 = 300;                % SiO2层厚度(nm)
```
## 核心函数说明
1. `sgl()` - 计算单层界面的反射率
2. `mul()` - 计算多层结构的反射率
3. `dwsurf()` - 绘制反射率分布图
4. `forTrans()` - 计算传递函数
5. `coordinate()` - 设置坐标系统
6. `polarition()` - 计算电场分量
7. `Fourier()` - 进行傅里叶变换处理
## 输出结果
程序将输出：
- 反射率随波长和入射角的变化图
- 最小反射率对应的波长和入射角
- 入射和反射电场分布图
- 相位分布图
